募集要項（PDF）
入居申込期間
2026年2月17日(火)〜2026年3月17日(日)
入居申込資格
  • 年齢が30代前半までの単身の方
  • クリエーティブな活動を生業とすることを目指している方で、ポートフォリオなど活動実績が確認できる資料を提出できる方
  • 他の入居者や地域住民と良好なコミュニケーションが取れる方
  • 地域活動（町会などの地域団体が主催する活動）への参加意欲がある方　など
居住の条件
  • 単身での居住
  • 日常生活で想定される範囲を超える音、臭い、振動の発生する作業等の禁止
  • クリエーターとしての創作活動の継続
  • 他の入居者や地域住民との交流、地域団体等の活動への積極的な参加・協力
  • 活動報告会への参加及び活動報告（年１回）　など
