Living

{ 暮らす }

創作活動が軌道に乗るまでのスタートアップ支援として、リーズナブルな家賃で住環境を提供します。クリエーターの経済的負担を軽減し、創作活動に打ち込むことができる環境を提供します。

Creating

{ つくる }

クリエーターに一定の創作活動が可能な環境を提供するほか、入居者同士や地域住民との交流を通じた、新たな価値を創造する「きっかけ」となる場を提供します。

Connecting

{ つながる }

クリエーターや地域住民、ボランティアなど「人と人」「人と地域」がつながり、交流する場として、併設の地域交流施設を中心に「地域の縁側」のような居心地の良い空間を、居住者と地域住民に提供します。
